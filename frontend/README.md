# SourceCast Frontend

The SourceCast web interface is built with Next.js and TypeScript. It provides
authentication, knowledge-space management, source ingestion, authenticated job
progress streaming, transcript browsing, and source-scoped evidence retrieval.

From this directory:

```powershell
Copy-Item .env.local.example .env.local
npm install
npm run dev
```

Use `npm run test`, `npm run lint`, and `npm run build` to verify frontend
changes.
