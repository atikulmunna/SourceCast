# SourceCast Frontend

The SourceCast web interface is built with Next.js and TypeScript. It provides
authentication, knowledge-space management, source ingestion, authenticated job
progress streaming, transcript browsing, evidence-grounded chat with session
resume, source comparison, saved insights, and Markdown research brief export.

From this directory:

```powershell
npm install
npm run dev
```

Use `npm run test`, `npm run lint`, and `npm run build` to verify frontend
changes.
